# Contributing

Thanks for your interest in OTP Bot! 🤖

## How to Contribute

1. Fork the repo
2. Create your feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Guidelines

- Keep credentials out of code — use env vars
- Add tests for new features when possible
- Follow existing code style (120 char line limit)
- Update README if you add/change functionality

## Reporting Issues

- Describe what you expected vs what happened
- Include Python version and `python-telegram-bot` version
- For bugs, minimal reproduction steps help a lot!
